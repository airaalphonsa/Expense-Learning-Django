# Expense-Learning-Django
A Django web application for learning and managing expenses, featuring a customized Django Admin panel and modern UI design.
# ExpenseLearning

ExpenseLearning is a beginner-friendly Django project created to learn and practice
web development using Django. The project focuses on basic CRUD operations and
Django admin usage.

---

## Features
- Django Admin Panel
- User Authentication
- Expense Management (Learning Purpose)
- Clean and simple project structure

---

## Requirements
- Python 3.x
- Django

---

## How to Run the Project

### Step 1: Clone or Download the Project
Download the project ZIP from GitHub and extract it.

### Step 2: Create Virtual Environment
```bash
python -m venv myenv
Step 3: Activate Virtual Environment

Windows

myenv\Scripts\activate

Step 4: Install Django
pip install django

Step 5: Run Migrations
python manage.py migrate

Step 6: Run Server
python manage.py runserver


Open browser and visit:

http://127.0.0.1:8000/

Create Superuser (Admin Login)

Run the following command:

python manage.py createsuperuser


Enter:

Username

Email

Password

Then open:

http://127.0.0.1:8000/admin/


Login using superuser credentials.
